Detailed Description

template<typename T>
struct fastuidraw::gl::opengl_trait< T >

Type trait struct that provides type information to feed GL commands.

Template Parameters
Ttype from which to extract values. The struct is specicialized for each of the GL types: GLubyte, GLbyte, GLuint, GLint, GLushort, GLshort, GLfloat and recursively for vecN
