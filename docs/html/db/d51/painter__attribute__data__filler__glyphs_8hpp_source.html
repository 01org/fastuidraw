36  * PainterAttributeDataFiller for the purpose of drawing glyphs.

38  * The enumeration glyph_type provide the indices into
39  * PainterAttributeData::attribute_data_chunks() and
40  * PainterAttributeData::index_data_chunks() for the different
41  * glyph types. It is an error for any of the valid glyphs passed
42  * to not be uploaded to the GlyphAtlas.
43  * Data for glyphs is packed as follows:
44  * - PainterAttribute::m_attrib0 .xy -> position in item coordinates (float)
45  * - PainterAttribute::m_attrib0 .zw -> the difference in item coordinates

48  * - PainterAttribute::m_attrib1 .x -> Glyph::attribute()[0]

52  * - PainterAttribute::m_attrib2 .x -> Glyph::attribute()[4]
53  * - PainterAttribute::m_attrib2 .y -> Glyph::attribute()[5]
54  * - PainterAttribute::m_attrib2 .z -> Glyph::attribute()[6]
