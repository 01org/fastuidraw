enum fastuidraw::glsl::PainterShaderRegistrarGLSLTypes::clipping_type_t

Enumeration specifying how the uber-shaders will perform clipping against the clip-planes of PainterClipEquations

Enumerator
clipping_via_gl_clip_distance 

Clipping is performed in the vertex-shader using gl_ClipDistance[i] for 0 <= i < 4.

clipping_via_discard 

Clipping is performed by passing the distance to each clip-plane and performing discard in the fragment shader.

clipping_via_skip_color_write 

Clipping is performed by passing the distance to each clip-plane and (virtually) skipping the color write. This is active if the active PainterBlendShader has PainterBlendShader::type() as PainterBlendShader::framebuffer_fetch. For other blend-types, this is the same as clipping_via_discard.

enum fastuidraw::glsl::PainterShaderRegistrarGLSLTypes::fbf_blending_type_t

Enumeration to specify how to perform framebuffer-fetch blending.

Enumerator
fbf_blending_not_supported 

Indicates that framebuffer-fetch blending is not supported.

fbf_blending_framebuffer_fetch 

Use framebuffer fetch (i.e. the out of the fragment shader is an inout).

fbf_blending_interlock 

Have the color buffer realized as an image2D and use fragment shader interlock to get blending order correct.
