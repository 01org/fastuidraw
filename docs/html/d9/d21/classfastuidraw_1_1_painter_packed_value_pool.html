Detailed Description

A PainterPackedValuePool can be used to create PainterPackedValue objects.

Just like PainterPackedValue, PainterPackedValuePool is NOT thread safe, as such it is not a safe operation to use the same PainterPackedValuePool object from multiple threads at the same time. A fixed PainterPackedValuePool can create PainterPackedValue objects used by different Painter (and PainterPacker) objects.
