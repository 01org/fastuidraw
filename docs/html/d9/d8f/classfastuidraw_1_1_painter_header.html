Detailed Description

A PainterHeader represents the values of the header for the shaders to read to draw data. The header holds data that is common for all vertices and fragments for an item to draw, including with what shader and the data for the shader.

drawing_occluder 

When m_blend_shader_data_location is set to drawing_occluder, it indicates that the item being drawing is an occluder thus the color computation from the brush and item can be skipped.

fixed_function_brush_shader 

Bit mask (with only the leading bit up). If the value of m_brush_shader bitwise anded with this value is non-zero, it indicates that the brush is realized by a PainterBrush. If the bitwise and value is zero, then it is realized by a PainterCustomBrushShader.

◆ m_blend_shader_data_location

uint32_t fastuidraw::PainterHeader::m_blend_shader_data_location

The location, in units of vecN<generic_data, 4> tuples, to the location in the data store buffer (PainterDraw::m_store) for the blend shader data. I.e. the PainterBlendShaderData value is stored (packed) at the location

PainterDraw::m_store[m_blend_shader_data_location * 4]

NOTE: if m_blend_shader_data_location is drawing_occluder this means that the item being drawing is an occluder thus the color computation from the brush and item can be skipped.
