Detailed Description

A PainterBrush defines a brush for painting via Painter.

The brush applies, in the following order:

  1. a constant color, specified by color()
  2. optionally modulates by an image, specified by image() or sub_image() and no_image()
  3. optionally modulates by a gradient, see linear_gradient(), radial_gradient(), sweep_gradient() and no_gradient().

An item shader's vertex stage provides the coordinate fed to the brush. That coordinate is procesed in the following order before it is fed to the image and gradient:

  1. an optional 2x2 matrix is applied, see transformation_matrix(), apply_matrix() apply_shear(), apply_rotate() and no_transformation_matrix()
  2. an optional translation is applied specified, see transformation_translate(), apply_matrix() apply_translate() and no_transformation_translation()
  3. an optional repeat window is applied, see repeat_window() and no_repeat_window().

◆ gradient_type_t

enum fastuidraw::PainterBrush::gradient_type_t

Enumeration to specify what kind of gradient is applied.

Enumerator
no_gradient_type 

Indicates that no gradient is present

linear_gradient_type 

Indicates that a linear gradient is present; a linear gradient is defined by two points p0 and p1 where the interpolate at a point p is the value t when p = p0 + (p1 - p0) * t

radial_gradient_type 

Indicates that a radial gradient is present; a radial gradient is defined by two circles C0 = Circle(p0, r0), C1 = Circle(p1, r1) where the interpolate at a point p is the time t when p is on the circle C(t) where C(t) = Circle(p(t), r(t)) where p(t) = p0 + (p1 - p0) * t r(t) = r0 + (r1 - r0) * t

sweep_gradient_type 

A sweep gradient is defined by a single point C, an angle theta (in radians), a sign S and a factor F. The angle theta represents at what angle the gradient starts, the point C is the center point of the sweep, the sign of S represents the angle orientation and the factor F reprsents how many times the gradient is to be repated. Precisely, the interpolate at a point p is defined as t_interpolate where

vec2 d = p - C;
float theta, v;
theta = S * atan(p.y, p.x);
if (theta < alpha )
{
theta += 2 * PI;
}
theta -= alpha;
v = (theta - angle) / (2 * PI);
t_interpolate = (S < 0.0) ? F * (1.0 - v) : F * v;

◆ spread_type_t

enum fastuidraw::PainterBrush::spread_type_t

Enumeration to specify how a value is interpreted outside of its natural range. For gradients the range is [0, 1] acting on its interpolate.

Enumerator
spread_clamp 

Clamp the value to its range, i.e. for a value t on a range [A, B] the value is clamp(r, A, B).

spread_mirror 

Mirror the value across the start of its range, i.e. for a value t on a range [A, B] the value is clamp(A + abs(t - A), A, B)

spread_repeat 

Repeat the value to its range, i.e. for a value t on a range [A, B] the value is A + mod(t - A, B - A)

spread_mirror_repeat 

Mirror repeat the value across the start of its range, i.e. for a value t on a range [A, B] the value is B - abs(mod(t - A, 2 * (B - A)) - (B - A))

◆ shader()

uint32_t fastuidraw::PainterBrush::shader ( void  ) const

Returns the brush shader ID which when tested against the bit masks from shader_masks tells what features are active in the brush. The shader is decoded as follows:

  • The value given by
    unpack_bits(image_filter_bit0, image_filter_num_bits, shader())
    is non-zero if an image is present and when is non-zero the value's meaning is enumerated by image_filter
  • The value given by
    unpack_bits(gradient_type_bit0, gradient_type_num_bits, shader())
    gives what gradient (if any) the brush applies as according to gradient_type_t
  • The value given by
    unpack_bits(gradient_spread_type_bit0, gradient_spread_type_num_bits, shader())
    gives the gradient spread pattern (if any) the brush applies as according to spread_type_t
  • If shader() & repeat_window_mask is non-zero, then a repeat window is applied to the brush.
  • If shader() & transformation_translation_mask is non-zero, then a translation is applied to the brush.
  • If shader() & transformation_matrix_mask is non-zero, then a 2x2 matrix is applied to the brush.
