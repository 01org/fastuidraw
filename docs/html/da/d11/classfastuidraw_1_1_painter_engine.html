◆ create_backend()

virtual reference_counted_ptr<PainterBackend> fastuidraw::PainterEngine::create_backend ( void  ) const
pure virtual

To be implemented by a derived class to create a Painter Backend object. All Painter Backend objects created by create_backend() from the same PainterEngine share the same

  • PainterShaderRegistrar (see painter_shader_registrar())
  • GlyphAtlas (see glyph_atlas())
  • ImageAtlas (see image_atlas())
  • ColorStopAtlas (see colorstop_atlas()) but are otherwise independent of each other.

Implemented in fastuidraw::gl::PainterEngineGL.
