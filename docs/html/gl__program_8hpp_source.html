39 /*!
40  * \brief
41  * Simple Shader utility class providing a simple interface to build
42  * GL shader objects using a glsl::ShaderSouce as its source code.
43  *
44  * The actual GL object creation is defferred to later, in
45  * doing so, one can create Shader objects from outside the main GL
46  * thread. Each of the following commands
47  * - compile_success()
48  * - compile_log()
49  * - name()
50  *
51  * triggers the GL commands to compile the shader if the shader has
52  * not been yet attempeted to be compiled. Hence one may only call
53  * these from outside the rendering thread if shader_ready() returns
54  * true. Moreover, a Shader may only be deleted from the GL rendering
55  * thread.
56  */

69  ~Shader();
