42 /*!
43  * \brief
44  * An TessellatedPath represents the tessellation of a Path
45  * into line segments and arcs.
46  *
47  * A single contour of a TessellatedPath is constructed from
48  * a single \ref PathContour of the source \ref Path. Each
49  * edge of a contour of a TessellatedPath is contructed from
50  * a single \ref PathContour::interpolator_base of the source \ref
51  * PathContour. The ordering of the contours of a
52  * TessellatedPath is the same ordering as the source
53  * \ref PathContour objects of the source \ref Path. Also,
54  * the ordering of edges within a contour is the same ordering
55  * as the \ref PathContour::interpolator_base objects of
56  * the source \ref PathContour. In particular, for each contour
57  * of a TessellatedPath, if an edge is closed, the closing edge
58  * is the last edge.
59  */
