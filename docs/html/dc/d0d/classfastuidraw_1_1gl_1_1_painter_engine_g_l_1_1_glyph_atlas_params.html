Set glyph_data_backing_store() to glsl::PainterShaderRegistrarGLSL::glyph_data_texture_array, i.e. to use a 2D texture array to store the glyph data. The depth of the array is set implicitely by the size given by GlyphAtlasBackingStoreBase::size(). NOTE: if either parameter is made negative, the call is ignored.

Parameters
log2_widthLog2 of the width of the 2D texture array

◆ use_texture_2d_array_store() [2/2]

GlyphAtlasParams& fastuidraw::gl::PainterEngineGL::GlyphAtlasParams::use_texture_2d_array_store ( void  )

Set glyph_data_backing_store() to glsl::PainterShaderRegistrarGLSL::glyph_data_texture_array, i.e. to use a 2D texture array to store the glyph data. The depth of the array is set implicitely by the size given by GlyphAtlasBackingStoreBase::size(). The width and height of the texture will be selected from examining the context properties.
