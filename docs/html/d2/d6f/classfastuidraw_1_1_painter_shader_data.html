Detailed Description

Common base class to PainterItemShaderData, PainterCompositeShaderData and PainterBlendShaderData to hold shader data for custom shaders.

Derived classes CANNOT add any data or virtual functions. The class PainterShaderData is essentially a wrapper over a PainterShaderData::DataBase object that handles holding data and copying itself (for the purpose of copying PainterShaderData objects).
