70  /*!
71  * \brief
72  * A PainterPackedValue represents a handle to an object that stores
73  * packed state data and tracks if that underlying data is already is
74  * already copied to PainterDraw::m_store.
75  *
76  * If already on a store, then rather than copying the data again, the data
77  * is reused. The object behind the handle is NOT thread safe. In addition
78  * the underlying reference count is not either. Hence any access (even dtor,
79  * copy ctor and equality operator) on a fixed object cannot be done from
80  * multiple threads simutaneously. A fixed PainterPackedValue can be used
81  * by different Painter objects, subject to that the access is done from
82  * same thread.
83  */
