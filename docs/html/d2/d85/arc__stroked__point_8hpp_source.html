32 /*!
33  * \brief
34  * An ArcStrokedPoint holds the data for a point of stroking
35  * where the segments can be arcs of a circle. The upshot
36  * is that a fragment shader will determine per-pixel coverage.
37  * In addition, the data is so that changing the stroking width
38  * or miter limit does not change the stroking data.
39  */
