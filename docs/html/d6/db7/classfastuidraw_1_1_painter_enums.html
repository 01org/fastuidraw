enum fastuidraw::PainterEnums::blend_w3c_mode_t

Enumeration specifying W3C blending modes.

Enumerator
blend_w3c_normal 

W3C multiply mode: Src, i.e. no blending operation

blend_w3c_multiply 

W3C multiply mode: Dest * Src

blend_w3c_screen 

W3C screen mode: 1 - (1 - Dest) * (1 - Src)

blend_w3c_overlay 

W3C overlay mode: for each channel, (Dst <= 0.5) ? 2.0 * Dest * Src : 1 - 2 * (1 - Dst) * (1 - Src)

blend_w3c_darken 

W3C darken mode: min(Src, Dest)

blend_w3c_lighten 

W3C lighten mode: max(Src, Dest)

blend_w3c_color_dodge 

W3C color-dodge mode: for each channel (Dest == 0) ? 0 : (Src == 1) ? 1 : min(1, Dst / (1 - Src) ) i.e. if Dest is 0, write 0. If Src is 1, write 1. Otherwise write Dst / (1 - Src)

blend_w3c_color_burn 

W3C color-burn mode: for each channel (Dest == 1) ? 1 : (Src == 0) ? 0 : 1 - min(1, (1 - Dst) / Src) i.e. if Dest is 1, write 1. If Src is 0, write 0. Otherwise write (1 - Dst ) / Src

blend_w3c_hardlight 

W3C hardlight mode: for each channel, (Src <= 0.5) ? 2.0 * Dest * Src : 1 - 2 * (1 - Dst) * (1 - Src)

blend_w3c_softlight 

W3C soft light mode: for each channel: (Src <= 0.5) ? Dst - (1 - 2 * Src) * Dst * (1 - Dst) : Dst + (2 * Src - 1) * (Z - Dst) where Z = (Dst <= 0.25) ? ((16 * Dst - 12) * Dst + 4) * Dst : sqrt(Dst)

blend_w3c_difference 

W3C difference mode: for each channel, abs(Dest - Src)

blend_w3c_exclusion 

W3C exclusion mode: for each channel, Dest + Src - 2 * Dest * Src

blend_w3c_hue 

w3c hue mode, see w3c for formula

blend_w3c_saturation 

w3c saturation mode, see w3c for formula

blend_w3c_color 

w3c color mode, see w3c for formula

blend_w3c_luminosity 

w3c luminosity mode, see w3c for formula

◆ shader_anti_alias_t

enum fastuidraw::PainterEnums::shader_anti_alias_t

Enumeration to specify the anti-aliasing quality to apply when performing path fills or strokes.

Enumerator
shader_anti_alias_none 

Do not apply any shader based anti-aliasing to the fill.

shader_anti_alias_simple 

Applies simpler anti-aliasing shading to path fill or stroke. This will potentially give under coverage to fragments (typically where the path crosses itself or when the path is filled or stroke highly minified).

shader_anti_alias_high_quality 

Applies higher quality anti-aliasing shading that avoids the issues that come from shader_anti_alias_simple. This option will give the highest quality anti-aliasing at the potential cost of performance.

shader_anti_alias_auto 

Represents to use shader_anti_alias_high_quality if using it does not have a large performance impact and otherwise to use shader_anti_alias_simple.

shader_anti_alias_fastest 

Represents to use the fastest anti-alias mode (which is not shader_anti_alias_none).

enum fastuidraw::PainterEnums::stroking_method_t

Enumeration to specify how to stroke

Enumerator
stroking_method_linear 

Use linear stroking taken directly from the Path. Thus the passed StrokedPath only consists of line segments.

stroking_method_arc 

Use arc-stroking, i.e. the passed StrokedPath has both arc-segments and line segments. This results in fewer vertices with the fragment shader computing per-pixel coverage.

stroking_method_auto 

Choose for optimal performance.
