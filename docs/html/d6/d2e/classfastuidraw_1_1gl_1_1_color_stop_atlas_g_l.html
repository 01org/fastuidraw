Detailed Description

An ColorStopAtlasGL is the GL(and GLES) backend implementation for ColorStopAtlas.

A ColorStopAtlasGL uses a GL texture for the underlying store. In GL, the texture type is GL_TEXTURE_1D_ARRAY, in GLES it is GL_TEXTURE_2D_ARRAY (because GLES does not support 1D texture).

The method flush() must be called with a GL context current. If the ColorStopAtlasGL was constructed as delayed, then the loading of data to the GL textures is delayed until flush, otherwise it is done immediately and then must be done with a GL context current.
