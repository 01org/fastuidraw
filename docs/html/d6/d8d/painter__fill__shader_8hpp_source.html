33  /*!
34  * \brief
35  * A PainterFillShader holds the shaders for drawing filled paths.
36  * Anti-aliasing is accomplished by drawing 1-pixel thick rects
37  * about the boundary of the filled path. The high quality method
38  * \ref PainterEnums::shader_anti_alias_high_quality renders the
39  * anti-alias pixels in two passes; the first pass to a coverage
40  * buffer and the second pass returns the value as the coverage
41  * and clears the auxilary buffer. The other renering method,
42  * \ref PainterEnums::shader_anti_alias_simple, is a single pass
43  * solution that relies on the depth buffer to avoid overdraw at
44  * the possible expense than some fragments (typically where the
45  * path crosses itself or when the path is drawn very minified)
46  * will have lower coverage than they should.
47  */
