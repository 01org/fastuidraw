85  /*!
86  * \brief
87  * A ColorStopArray is a sequence of ColorStop values used to
88  * define the color stops of a gradient.
89  *
90  * The values are sorted by ColorStop::m_place and each ColorStop
91  * value of a ColorStopArray must have a unique value for
92  * ColorStop::m_place. A color is computed (in drawing) from a
93  * ColorStopArray at a point q as follows. First the color stops
94  * S and T are found so that q is in the range [S.m_place, T.m_place].
95  * The color value is then given by the value
