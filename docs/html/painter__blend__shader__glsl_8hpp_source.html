36  * A PainterBlendShaderGLSL is a PainterBlendShader whose

38  *
39  * The code to implement is dependent on the PainterBlendShader::type()
40  * of the created PainterBlendShaderGLSL.
41  * - PainterBlendShader::type() == PainterBlendShader::single_src
42  * The shader code fragment must provide the function
43  * \code
44  * void

46  * in vec4 in_src, out vec4 out_src)
47  * \endcode
48  * where in_src is the output of the item fragment shader modulated by the
49  * current brush with alpha applied to rgb and out_src is the value for the
50  * fragment shader to emit.
51  *
52  * - PainterBlendShader::type() == PainterBlendShader::dual_src
53  * The shader code fragment must provide the function
54  * \code
55  * void
56  * fastuidraw_gl_compute_blend_factors(in uint sub_shader, in uint blend_shader_data_location,
57  * in vec4 in_src, out vec4 out_src0, out vec4 out_src1)
58  * \endcode
59  * where in_src is the output of the item fragment shader modulated by the
60  * current brush with alpha applied to rgb, out_src0 is the value for the
61  * fragment shader to emit for GL_SRC_COLOR and out_src1 is the value for
62  * the fragment shader to emit value for GL_SRC1_COLOR.
63  *
64  * - PainterBlendShader::type() == PainterBlendShader::framebuffer_fetch
65  * The shader code fragment must provide the function

70  * \endcode
71  * where in_src is the output of the item fragment shader modulated by the
72  * current brush with alpha applied to rgb, in_fb is the value of the
73  * framebuffer at the location and out_src is the value for the fragment
74  * shader to emit.
75  *
76  * For each of the blend shader type:
77  * - sub_shader corresponds to PainterBlendShader::sub_shader(),
