Detailed Description

A PainterEffect represents the interface to define and effect to apply to image data. At its core, it is made up of a sequence of passes. A derived class is expected to expand on PainterEffect to allow to specify properties of an effect (for example blur properties for a blur effect) and to reimplment copy().
