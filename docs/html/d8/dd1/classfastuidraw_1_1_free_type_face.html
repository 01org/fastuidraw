Detailed Description

A FreeTypeFace wraps an FT_Face object of the FreeType library together with a mutex in a reference counted object.

The threading model for the FreeType appears to be:

  • Create an FT_Library object
  • When creating or releasing FT_Face objects, lock a mutex around the FT_Library when doing so
  • If an FT_Face is accessed from multiple threads, the FT_Face (but not the FT_Library) needs to be mutex locked
