Detailed Description

GlyphAtlasStoreBase represents an interface to an aray of generic_data values.

An example implementation in GL would be a buffer object that backs a single usamplerBuffer. An implementation of the class does NOT need to be thread safe because the ultimate user of the backing store (GlyphCache) performs calls to the backing store behind its own mutex.
