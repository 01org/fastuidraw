Detailed Description

Base class to hold custom data for custom brush shaders.

Derived classes CANNOT add any data or virtual functions. The class PainterCustomBrushShaderData is essentially a wrapper over a PainterCustomBrushShaderData::DataBase object that handles holding data and copying itself (for the purpose of copying PainterCustomBrushShaderData objects).
