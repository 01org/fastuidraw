c_string  function_name,

Stream the unpack function into a ShaderSource object. For values constructed passed a single c_string, the function generated is

return_type
function_name(in uint location, out struct_name v)

and for those values constructed passed an array of c_string values, the function generated is

return_type
function_name(in uint location, out struct_name0 v0, out struct_name1 v1, ...)

where for voth, location is the location of the struct to unpack, struct_name is the name of the struct to unpack to given in the ctor.

Parameters
strShaderSource to which to stream the unpack function
function_namename to give the function
returns_new_offsetif true, the return_type is an uint and returns the offset of the last offset supplied by set(). If value, return_type is void.
