const reference_counted_ptr<PainterItemShader>& fastuidraw::PainterFillShader::aa_fuzz_deferred_coverage ( void  ) const

Returns the PainterItemShader to use to draw the anti-alias fuzz via the deferred coverage buffer. This shader will draw the aa-fuzz in two passes: the first pass draws the coverage as computed by a fragment shader to the deferred coverage buffer (via the PainterItemShader::coverage_shader()) and the 2nd pass reads from the deferred coverage buffer to emit the alpha value.

◆ aa_fuzz_immediate_coverage_pass1() [1/2]

const reference_counted_ptr<PainterItemShader>& fastuidraw::PainterFillShader::aa_fuzz_immediate_coverage_pass1 ( void  ) const

Returns the PainterItemShader to use to draw the 1st pass of the anti-alias fuzz via the immediate coverage buffer. The first pass renders a coverage value to the immediate coverage buffer and the 2nd pass reads the coverage value from the immediate coverage buffer and clears it. The expected format of the attributes is as found in the PainterAttributeData returned by FilledPath::Subset::aa_fuzz_painter_data().
