Detailed Description

Class for creating and using GLSL programs.

A Program delays the GL commands to create the actual GL program until the first time it is bound with use_program(). In addition to proving the GL code to create the GLSL code, Program also provides queries GL for all active uniforms and attributes (see active_attributes(), active_uniforms(), find_uniform() and find_attribute()). Also, provides an interface so that a sequence of GL commands are executed the first time it is bound and also an interface so a sequence of actions is executed every time it is bound. Program's are considered a resource, as such have a resource manager.

enum fastuidraw::gl::Program::shader_variable_src_t

Enumeration to describe the backing of a shader variable's value.

Enumerator
src_default_uniform_block 

Indicates that the shader variable is from the default uniform block; the variable's value is not sourced from a backing buffer object.

src_abo 

Indicats that the shader variable is an atomic buffer counter and sourced from a backing buffer object.

src_uniform_block 

Indicats that the shader variable is from the a uniform block; the variable's value is sourced from a backing buffer object.

src_shader_storage_block 

Indicats that the shader variable is from the a shader storage block; the variable is backed by a buffer object.

src_shader_input 

Indicates that the shader variable is an input (i.e an 'in' of GLSL).

src_shader_output 

Indicates that the shader variable is an output (i.e an 'out' of GLSL).

src_shader_transform_feedback 

Indicates that the shader variable is a transform feedback variable

src_null 

Indicates that the shader variable is a nullptr value; such values are returned when a query for a shader variable is made and there is no such shader variable.

Returns the number of active transform feedbacks. Note that an array of transform feedback is classified as a single element.
