  • interval_id (output) ID of interval

  • interval_end (output) interval end of interval

That compute the coverage from glyph data as packed by GlyphRenderDataRestrictedRays. The variant with additional out arguments also outputs in those additional out arguments various data values that are computed in doing the computation that a debug version of a shader can visualize the structure of the glyph. The returned ShaderSource also includes a large number of utility functions and struct each prefixed with fastuidraw_restricted_rays_ that assist in implementing the fastuidraw_restricted_rays_compute_coverage() functions.
