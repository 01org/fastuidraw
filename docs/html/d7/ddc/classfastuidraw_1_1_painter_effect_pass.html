Detailed Description

A PainterEffectPass represents the interface to define a single pass of a PainterEffect. A pass is essentially a mutable brush changed by what image to which it will be applied. A single PainterEffectPass is to be used by only one thread at a time.

pure virtual

To be implemented by a derived class to return the brush made from the passed Image value. The returned PainterData::brush_value value needs to be valid until the PainterEffectPass dtor is called or the next all to brush(). The passed image is guaranteed to have Image::type() as value Image::bindless_texture2d or Image::context_texture2d.
