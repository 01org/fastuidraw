Detailed Description

A PainterEngineGL implements PainterEngine using the GL (or GLES) API. The PainterShaderRegistrar object returned by painter_shader_registrar() derives from glsl::PainterShaderRegistrarGLSL. In addition, when adding custom shaders, the shader types need to be derived from the shader types of the GLSL module, i.e.

  • Item shaders must be realized as glsl::PainterItemShaderGLSL objects or objects derived from glsl::PainterItemShaderGLSL
  • Item coverage shaders must be realized as glsl::PainterItemCoverageShaderGLSL objects or objects derived from glsl::PainterItemCoverageShaderGLSL
  • Blend shaders must be realized as glsl::PainterBlendShaderGLSL objects or objects derived from glsl::PainterBlendShaderGLSL
  • Custom brush shaders must be realized as glsl::PainterBrushShaderGLSL objects or objects derived from glsl::PainterBrushShaderGLSL
