36  /*!
37  * \brief
38  * A PainterBrush defines a brush for painting via Painter.
39  *
40  * The brush applies, in the following order:
41  * -# a constant color, specified by \ref pen()
42  * -# optionally applies an image, specified by \ref image() or
43  * \ref sub_image() also see \ref no_image()
44  * -# optionally applies a linear or radial gradient, see \ref
45  * linear_gradient() and \ref radial_gradient() also see
46  * \ref no_gradient().
47  *
48  * An item shader's vertex stage provides the coordinate
49  * fed to the brush. That coordinate is procesed in the
50  * following order before it is fed to the image and gradient:
51  * -# an optional 2x2 matrix is applied specified by \ref
52  * transformation_matrix(), also see \ref
53  * no_transformation_matrix() and \ref
54  * transformation().
55  * -# an optional translation is applied specified by \ref
56  * transformation_translate(), also see \ref
57  * no_transformation_translation() and \ref
58  * transformation().
59  * -# an optional repeat window is applied specified by \ref
60  * repeat_window(), also see no_repeat_window().
61  */
62  class PainterBrush
63  {
64  public:
