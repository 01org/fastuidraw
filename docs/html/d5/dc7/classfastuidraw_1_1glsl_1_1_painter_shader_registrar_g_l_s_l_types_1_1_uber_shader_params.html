If true, provide an image2D (of type r8) uniform to which to write coverage value for multi-pass shaders (in particular shader based ant-aliased stroking). Writing to the buffer should not be done, instead one should use the functions:

◆ use_ubo_for_uniforms() [1/2]

bool fastuidraw::glsl::PainterShaderRegistrarGLSLTypes::UberShaderParams::use_ubo_for_uniforms ( void  ) const

If true, use a UBO to back the uniforms of the uber-shader. If false, use an array of uniforms instead. The name of the UBO block is fastuidraw_uniforms and the name of the uniform is fastuidraw_uniforms. In both cases, the buffer can be filled by the function PainterShaderRegistrarGLSL::fill_uniform_buffer(). For the non-UBO case, the uniforms are realized as an array of floats in GLSL.
