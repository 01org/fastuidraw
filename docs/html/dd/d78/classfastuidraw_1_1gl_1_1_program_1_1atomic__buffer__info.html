GLint fastuidraw::gl::Program::atomic_buffer_info::buffer_binding ( void  ) const

The GL API index for the binding point of the atomic buffer, i.e. the value to feed as bufferIndex to the GL API functions:

glBindBufferBase(GL_ATOMIC_COUNTER_BUFFER, bufferIndex, ...)
glBindBufferRange(GL_ATOMIC_COUNTER_BUFFER, bufferIndex, ..)
GLint fastuidraw::gl::Program::atomic_buffer_info::buffer_index ( void  ) const

GL API index for querying the atomic buffer. The value of buffer_index() is used in calls to GL to query the atomic buffer block, i.e. the value to feed as index to the GL API functions:

glGetProgramResourceiv(program, GL_ATOMIC_COUNTER_BUFFER, index, ...)
glGetActiveAtomicCounterBufferiv(GLuint program, GLuint index, ...)

It is also the value to feed to Program::atomic_buffer().

GLint fastuidraw::gl::Program::atomic_buffer_info::buffer_size ( void  ) const

Returns the size in bytes of the atomic buffer (i.e. the size needed for a buffer object to correctly back the atomic buffer block).

c_string fastuidraw::gl::Program::atomic_buffer_info::name ( void  ) const

Block label (to match better interface of block_info); name is given as "#X_atomic_buffer" where X is the value of buffer_binding().

unsigned int fastuidraw::gl::Program::atomic_buffer_info::number_variables ( void  ) const

Returns the number of atomic -variables- of the atomic buffer. Note that an array is classified as a single variable.

fastuidraw::gl::Program::atomic_buffer_info::operator bool ( ) const

Implicit cast operator to bool. If returns false, indicates that the atomic_buffer_info is null, and thus buffer_index() returns -1, buffer_size() returns 0 and so on.
